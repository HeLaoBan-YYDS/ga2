## 2、网站主题色建议

结论：**默认用亮色主题**。Grow a Garden 2 是 Roblox 农场、种植、轻松休闲向主题，亮色更适合首页、Wiki、Guide、Codes 页面，视觉上更接近游戏本身的草地、种子、阳光氛围。

```css
/* 导航页主题色 - 亮色主题 */
--nav-theme: 142 62% 45%;        /* leaf green，接近 #2cb866 */
--nav-theme-light: 142 68% 58%;  /* 更浅的嫩绿色 */

/* 导航页主题色 - 暗色主题 */
--nav-theme: 142 62% 45%;        /* leaf green */
--nav-theme-light: 142 68% 58%;  /* 更浅的嫩绿色 */
```

可选辅助色：

```css
--nav-accent: 39 92% 56%;        /* seed yellow / orange */
--nav-bg-light: 48 100% 97%;     /* warm farm background */
--nav-bg-dark: 155 32% 10%;      /* deep garden green */
```

## 3、多语言建议

结论，最多 4 门，按优先级：

1. **English**
   本地化主题名：`Grow a Garden 2 Wiki`、`Grow a Garden 2 Codes`、`Grow a Garden 2 Beginner Guide`
   原因：Roblox、YouTube、Discord、Reddit 主搜索需求主要是英文，首发内容必须做英语。

2. **Spanish / Español**
   本地化主题名：`Códigos de Grow a Garden 2`、`Guía de Grow a Garden 2`
   原因：西语区已经出现 “códigos / Roblox / Grow a Garden 2” 搜索与视频内容，适合做 codes、guía、semillas 页面。([YouTube][4])

3. **Portuguese Brazil / Português Brasil**
   本地化主题名：`Códigos Grow a Garden 2`、`Guia Grow a Garden 2`
   原因：巴西 Roblox 用户量大，葡语内容里已经出现 Grow a Garden 2、códigos、guia、early access 等本地化表达。([GAMES.GG][5])

4. **Thai / ไทย**
   本地化主题名：`โค้ด Grow A Garden 2`、`คู่มือ Grow a Garden 2`
   原因：泰语区对 Roblox 热门游戏响应很快，搜索结果里已经出现泰语 “โค้ด / รหัสเกม / Grow A Garden 2” 相关内容，适合做 codes 和 beginner guide。([online-station.net][6])
